# zeppos-jira-workflow

## 安装方法
1. 默认安装到当前目录下的 zeppos-jira-workflow/
sh -c "$(curl -fsSL https://raw.githubusercontent.com/active-ailab/zeppos-jira-workflow/main/install.sh)"

2. 指定安装到 /path/to/parent/zeppos-jira-workflow/
sh -c "$(curl -fsSL https://raw.githubusercontent.com/active-ailab/zeppos-jira-workflow/main/install.sh)" --path XXXX/
